# BantayPresyo
 
This project is a backend tool for BantayPresyo for processing Department of Agriculture (DA) price monitoring PDFs into a normalized SQLite database. It extracts daily price data, maps commodities to their aliases via a `translations.json` file, and stores them in a schema designed for historical analysis. The system validates incoming PDFs for required sections, performs database maintenance to retain only the last 365 days of pricing, cleans up orphaned records, and generates a `version.json` file for frontend synchronization. It is designed to transform unstructured price report files into a structured, queryable format for trend tracking and price monitoring applications.
